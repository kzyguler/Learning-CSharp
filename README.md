# C# If-Else Practice

A beginner-friendly C# console application created to practice conditional statements and strengthen core programming fundamentals.

## About

This project was developed as part of my journey to becoming a Game Developer. It focuses on understanding how decision-making works in C# using conditional statements.

## Features

- Variable declarations
- Console input and output
- `if` statements
- `else if` statements
- `else` statements
- Comparison operators (`>`, `<`, `>=`, `<=`, `==`, `!=`)
- Simple ranking system example

## Technologies

- C#
- .NET
- Visual Studio 2022
